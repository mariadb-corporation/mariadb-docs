# MariaDB Enterprise Kubernetes Operator 26.03.1

**Release date**: 13 March 2026

If you are running in a version prior to 26.03.x, **please follow the [UPDATE GUIDE](https://mariadb.com/docs/tools/mariadb-enterprise-operator/updates/update-26.03)** to ensure a safe transition.

This is a patch release which includes the following fixes:
- __K8SDKR-168__:  Transparent Data Encryption (TDE) is now compatible with physical backups managed by the `PhysicalBackup` resource. The operator propagates the `env`, `volumes` and `volumeMounts` from the `MariaDB` resource to the `PhysicalBackup` jobs. This allows the backup jobs to have access to the TDE key and the `VAULT_TOKEN` environment variable, so mariadb-backup can function normally with TDE enabled.
