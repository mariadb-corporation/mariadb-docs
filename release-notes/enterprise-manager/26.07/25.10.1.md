# MariaDB Enterprise Manager 26.07.1 Release Notes

**Release Date:** 16 Jul 2026\
\
Release 26.07.1 is a GA release.&#x20;

This document describes the changes in release 26.07.1, when compared to the previous release in the same series.

### Bug Fixes

* MCDEV-4136 Enterprise Manager server fails to install on ARM64 architecture.

### Packaging <a href="#packaging" id="packaging"></a>

* Enterprise Manager is delivered as a suite of container images (via MariaDB Enterprise Docker Registry).
* Agent is delivered via the `mema-agent` OS package via RPM/DEB packages (via MariaDB Enterprise Tools repository).
