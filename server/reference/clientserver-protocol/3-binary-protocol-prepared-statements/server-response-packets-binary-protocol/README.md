# Server Response Packets (Binary Protocol)

The following response packets will be sent from the server to client in binary protocol only. They may be mixed with packets from the text protocol, like [OK\_PACKET](https://kb-archive.mariadb.net/kb/en/ok_packet/), `PACKET_METADATA` or [ERR\_PACKET](https://kb-archive.mariadb.net/kb/en/err_packet/).
