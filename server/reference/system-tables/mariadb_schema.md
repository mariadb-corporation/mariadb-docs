# mariadb\_schema

This page has been moved. See [Schema Qualifiers](../sql-structure/sql-language-structure/schema-qualifiers.md).

The `mariadb_schema` and `oracle_schema` prefixes are namespace qualifiers (prefixes) for data types and functions, rather than system tables. Both prefixes are now documented together to provide a better overview of SQL mode compatibility.
